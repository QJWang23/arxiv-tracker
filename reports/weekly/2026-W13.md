### PackForcing: Short Video Training Suffices for Long Video Sampling and Long Context Inference

**来源**: arxiv | **日期**: 2026-03-26 | **热度**: 20 | **层级**: Tier1 关键 | **标签**: inference-opt, long-context, hw-accel

**核心要点**
- **三分区 KV-cache 策略**：Sink tokens（保留早期锚帧全分辨率）+ Mid tokens（32x 时空压缩）+ Recent tokens（全分辨率局部连贯），实现有界内存占用
- **动态 top-k 上下文选择** + **连续 Temporal RoPE Adjustment**：无开销地重新对齐丢弃 token 导致的位置间隙
- 单张 H200，4GB KV-cache，生成 2 分钟 832x480@16fps 视频，24x 时序外推（5s→120s）

**创新点**
首次实现用 5 秒短视频训练直接推广到 120 秒长视频生成。核心思想是将 KV-cache 按语义重要性分层压缩，而非均匀丢弃，在严格内存约束下保持生成质量。

**对关注领域的影响**
- **推理优化**: 三分区 KV-cache 策略可直接迁移到 LLM 长上下文推理，解决线性增长的 KV-cache 内存瓶颈
- **硬件加速**: 32x 压缩通过 3D 卷积 + 低分辨率 VAE 实现，显存占用从无限增长收敛到 4GB 上界
- **K8s 基础设施**: 单 GPU 长视频生成降低分布式部署复杂度，适合容器化弹性调度

**后续跟进**
- [ ] 对比 StreamingLLM rolling cache 与 PackForcing 三分区策略的注意力稀疏化效果
- [ ] 评估分层 KV-cache 压缩策略在 vLLM serving 中的可行性
- [ ] 代码: https://github.com/ShandaAI/PackForcing

### FreeLOC: Free-Lunch Long Video Generation via Layer-Adaptive O.O.D Correction

**来源**: arxiv | **日期**: 2026-03-26 | **热度**: 20 | **层级**: Tier1 关键 | **标签**: long-context

**核心要点**
- **两种 O.O.D 问题识别**：帧级相对位置 O.O.D 和上下文长度 O.O.D 是长视频质量下降的根因
- **VRPR（Video-based Relative Position Re-encoding）**: 多粒度层次化重新编码时间相对位置，对齐预训练分布
- **TSA（Tiered Sparse Attention）**: 结构化不同时间尺度的注意力密度，兼顾局部细节和长程依赖
- **Layer-adaptive probing**: 自动识别各 transformer 层对 O.O.D 的敏感度，选择性应用修正

**创新点**
无需任何额外训练即可提升长视频生成质量。关键洞察是通过 layer-adaptive 机制识别哪些层需要修正、哪些层不受 O.O.D 影响，避免全层修正带来的额外开销。

**对关注领域的影响**
- **推理优化**: Training-free 特性使部署成本为零；layer-adaptive 思想可用于 LLM 推理中识别哪些层需要额外计算资源
- **硬件加速**: TSA 稀疏注意力减少长上下文的计算量

**后续跟进**
- [ ] 评估 layer-adaptive probing 思想在 LLM 推理中的 KV-cache 分配优化
- [ ] 代码: https://github.com/Westlake-AGI-Lab/FreeLOC

### S2D2: Fast Decoding for Diffusion LLMs via Training-Free Self-Speculation

**来源**: arxiv | **日期**: 2026-03-26 | **热度**: 14 | **层级**: Tier2 重要 | **标签**: inference-opt, hw-accel

**核心要点**
- **Self-Speculative Decoding**: 同一扩散模型在 block size=1 时退化为自回归模型，因此可同时充当 drafter 和 verifier
- **轻量路由策略**: 决定何时值得进行验证步骤，形成 diffusion 并行提议 + autoregressive 序列级验证的混合解码轨迹
- 在 SDAR 上实现 4.7x 自回归加速，相比 tuned baseline 提升 1.57x 且准确率 +4.5 分

**创新点**
首次在 block-diffusion LLM 中实现训练自由的投机解码。核心发现是模型自身的 block=1 模式可作为验证器，无需额外的 draft model。

**对关注领域的影响**
- **推理优化**: 为 diffusion LLM 推理加速提供了新范式，投机解码无需额外模型
- **硬件加速**: 减少 denoising steps 直接降低 GPU 计算量

**后续跟进**
- [ ] 对比 S2D2 与传统 speculative decoding 在延迟和吞吐上的差异
- [ ] 研究是否可将 self-speculation 思想应用到标准 LLM 的 speculative decoding

## 值得关注的量化论文

### GlowQ: Group-Shared Low-Rank Approximation for Quantized LLMs

**热度**: 8 | **标签**: inference-opt

**核心要点**: 提出组共享低秩近似方法，为量化 LLM 缓存单个共享右因子，避免在每个 decoder block 插入纠错模块。相比 LQER/QERA/ASER 等方法显著降低延迟和内存开销。

**后续跟进**: [ ] 评估 GlowQ 在 INT4 量化 vLLM serving 中的准确性和延迟表现

### SliderQuant: Accurate Post-Training Quantization for LLMs

**热度**: 8 | **标签**: inference-opt, architecture

**核心要点**: 发现浅层/深层对量化更敏感，中间层更鲁棒。基于此提出分层量化策略，为不同层分配不同比特宽度，在 3-4 bit 设置下显著优于均匀量化。

**后续跟进**: [ ] 对比 SliderQuant 与 AWQ/GPTQ 在实际部署中的效果
